## ğŸ“ Problem Statement (Brief)

Many chatbots give generic answers and cannot provide accurate, document-specific information. Users need a system that can answer questions based on their own documents.

This project builds a Retrieval-Augmented Generation (RAG) app using LangChain and Streamlit, allowing users to:

- Upload documents.  
- Convert them into a searchable vector database.  
- Ask questions and get answers grounded in the uploaded content.  
- Interact through an easy-to-use web interface with streaming responses.  

It combines document retrieval + LLM generation to provide context-aware, accurate responses.
